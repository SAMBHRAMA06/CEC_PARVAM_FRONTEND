<h1> - Heading

Defines a top-level heading.

<h1>Welcome to My Page</h1>

<p> - Paragraph

Used to write blocks of text.

<p>This is a paragraph.</p>

<a> - Anchor

Creates a hyperlink.

<a href="https://example.com" target="_blank">Visit Site</a>

<img> - Image

Displays an image.

<img src="image.jpg" alt="My Image" width="200">

<ul> - Unordered List

Creates a bullet-point list.


<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
