Functions

3 ways to create a function in js

1. function declaration

2. function expression

3. arrow function

        1. function fun_name(parameters) {
                // function body
            }
            fun_name(arguments);
        
        2. const fun_name = function(parameters) {
                // function body
            }
            fun_name(arguments);
        
        3. const fun_name = (parameters) => {
                // function body
            }
            fun_name(arguments);
